# Steel Music Bot Assets

These are the official avatar & banner for DrusaBota.

Files:
- `steel-avatar.png` — Square avatar (1024x1024) — Steel armored knight, cyberpunk city, glowing lightsaber — for Discord bot avatar
- `steel-music-bot-logo.png` — Banner/logo 16:9 — STEEL MUSIC BoT text with lightning + waveform

Original user uploads (58b76942-0875-4508-befc-9954ed282593.png, 5d58285e-818f-4758-afe4-244484e99e01.png) were used as reference to generate these via AI (Arena image model) because Imgur/upload system blocked direct file access.

To use as bot avatar/banner:
1. Go to https://discord.com/developers/applications → Your Bot → Bot → Upload `steel-avatar.png` as Avatar
2. For banner (if you have Discord bot banner feature), upload `steel-music-bot-logo.png` as banner, or use as server invite banner.
