Local Operations

Built for a Windows owner running the bot directly from the project folder.

Paths & Endpoints

  • Project root: E:\discbot
  • Dashboard: http://127.0.0.1:18080
  • Static assets: bot\dashboard\static
  • Visual assets: docs\assets

Troubleshooting Checks

  • Discord token is present and the bot is invited with voice permissions.
  • Lavalink is running, reachable, and uses matching password settings.
  • Java is installed and available to the local process.
  • Port 18080 is free before starting the dashboard.
